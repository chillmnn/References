# CHMOD

RWX | RWX | RWX

| Digit | Meaning |
|-------|---------|
| 1 |	File can be executed |
| 2	| File can be written to |
| 3 | File can be executed and written to |
| 4 | File can be read |
| 5 | File can be read and executed |
| 6 | File can be written to and read |
| 7 | File can be read, written to, and executed |
